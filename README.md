## 제출 방법

1. 자신의 깃허브 아이디로 브랜치 만든 다음에 개인 폴더 만들어주세요.
2. 작업 하시고 커밋 후 오리진으로 푸시해주시면 됩니다.
3. main 으로 풀리퀘 날려주시면 됩니다. merge 는 하시면 안돼요! 풀리퀘 제목 : 3기_10주차_[이름]

### 예시

```bash
git clone git@github.com:HateSlop/3-Agent-2.git  # 클론
cd 3-Agent-2 # 프로젝트 루트로 이동
git checkout -b '본인 브랜치명' # 브랜치 생성 (본인의 브랜치, 폴더 등 생성)
mkdir '본인 폴더명' # 개인 폴더 만들기
cd '본인 폴더명' # 개인 폴더로 이동
# 작업을 진행해주세요
git add . # 작업 후 add
git commit -m "[feat] ~~" # 커밋
git push origin '본인 브랜치명' # 오리진에 푸시
```
